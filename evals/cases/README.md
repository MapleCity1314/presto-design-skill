# Eval Cases

This folder stores one file per eval case so the suite is easier to scan and edit.

## Files

- [`01-redesign-saas-landing.json`](./01-redesign-saas-landing.json): redesign an existing SaaS landing page without changing the stack
- [`02-create-dashboard-tool.json`](./02-create-dashboard-tool.json): create a new SaaS landing page from scratch
- [`03-refine-typography-color.json`](./03-refine-typography-color.json): refine typography, color, and card treatment without changing layout structure

## Editing rule

When you add or update a case here, mirror the same change in [`../evals.json`](../evals.json) until the harness reads case files directly.
